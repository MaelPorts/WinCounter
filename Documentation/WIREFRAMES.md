## Color Scheme Suggestions

### Option 1: Classic FIFA

- **Background**: Dark navy blue (#0E1428)
- **Primary**: Bright green (#00FF87)
- **Secondary**: White (#FFFFFF)
- **Accent**: Gold (#FFD700)
- **Player 1 Button**: Blue (#3498DB)
- **Player 2 Button**: Red (#E74C3C)

### Option 2: Clean Modern

- **Background**: White (#FFFFFF)
- **Primary**: Dark gray (#2C3E50)
- **Secondary**: Light gray (#ECF0F1)
- **Accent**: Green (#27AE60)
- **Player 1 Button**: Teal (#1ABC9C)
- **Player 2 Button**: Orange (#E67E22)

### Option 3: Dark Mode

- **Background**: Black (#000000)
- **Primary**: White (#FFFFFF)
- **Secondary**: Dark gray (#34495E)
- **Accent**: Purple (#9B59B6)
- **Player 1 Button**: Cyan (#00D9FF)
- **Player 2 Button**: Magenta (#FF00D9)

---

## Responsive Design Notes

### Mobile (Portrait)

- Stack player stats vertically
- Make buttons full width
- Increase button size for touch
- Simplify stats to W/L/D on single line

### Tablet/Desktop

- Side-by-side layout as shown above
- Larger buttons with hover effects
- More spacing between elements
- Optional: Show match history sidebar

---

## Interactive Elements

### Buttons

- **Hover**: Slight scale increase (1.05x)
- **Active/Click**: Scale down (0.95x)
- **Transition**: 150ms ease
- **Feedback**: Brief color flash or ripple effect

### Stats Update Animation

- **Number Change**: Fade out old â†’ fade in new
- **Duration**: 200ms
- **Easing**: Ease-in-out

### Last Game Message

- **Update**: Slide in from bottom or fade in
- **Duration**: 300ms
- **Include**: Confetti or celebration animation for wins
